#COUNTRY

WHAT:
The countryside features 9 active locations you could visit with 4 characters found-throughout. There are 2 mini-games featured here. Menu not featured, and Sun is navigable point returning user to map to select other locations for travel.
WHY:
Flash gaming was pretty popular in the mid-late 2000s, and it was incredibly easy to create. From learning simple mechanics as projects, it slowly grew into a game itself.
HOW:
Role-playing game built in Flash via SWiSH MAX 2. Graphics were created within SWiSH MAX.
WHEN:
2009-2010
