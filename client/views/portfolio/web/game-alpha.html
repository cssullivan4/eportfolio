#ALPHA VERSION

WHAT:
Original version of the world & characters. The game grew from these established characters and sets. Each set features the character introducing themselves or acting in such a way that becomes for what they are later known.
WHY:
Flash gaming was pretty popular in the mid-late 2000s, and it was incredibly easy to create. From learning simple mechanics as projects, it slowly grew into a game itself.
HOW:
Role-playing game built in Flash via SWiSH MAX 2. Graphics were created within SWiSH MAX.
WHEN:
2009
