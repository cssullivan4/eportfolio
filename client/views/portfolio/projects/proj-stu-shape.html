#SHAPES GENERATOR

>>> IDEA

Create application that generates shapes based on customizable dimensions.

>>> FEATURES

  • HTML, CSS, JavaScript
  • Shapes generate within defined area
  • Click on shape to display its properties in side panel
  • Double-click shape to delete it
