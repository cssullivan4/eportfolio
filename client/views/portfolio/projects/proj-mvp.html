#STUDENT PROJECT

>>> PROMPT

Create web app to be deployed and presented. No specifics given to what the build must contain, but that it must be forked.

>>> TIMEFRAME

2 weeks to deploy & 1 week to prepare for presentation

>>> IDEA

Create portal for volunteers at the McKamey Animal Center to share their observations of the animals for other volunteers and staff. This project would allow volunteers to effectively access all animals available for adoption. Keeping in mind the use would most likely occur at the center on phones, specific information is available for quick access. The site was built responsively, as well.

>>> TEAM MEMBERS

Rachael Adams, Patricia Bazemore, Will Hearn, C. Samuel Sullivan

>>> FEATURES

  • 3 views (visitor, user session, admin session)
  • Search animal by shelter ID or any identifier
  • Comment, Delete, Flag on animal pages
  • Create, Edit, Delete, View profiles
  • Search user by any keyword or identifier
  • Donations page, powered by Stripe & SendGrid

>>> ROLE

Front-end, Navigation/Footer toggle controller, UX, Lead architecture, Miscelleaneous collaboration throughout

>>> APPROACH

Agile development using Trello, beginning an architecture led by me + several hand-drawn "wireframes." Team met goal of deploying fully-functional web-application within timeframe.

>>> TECHNOLOGIES

  • AngularJS, ExpressJS, jQuery, NodeJS, Bootstrap, Stripe, SendGrid, MySQL, Amazon Web Services
