"The Covalence Online Store is the final, instructional student project for the Covalence Full-Stack Development Program. "

#STUDENT PROJECT

>>> PROMPT

Create web app to be deployed and presented. Must create replica of provided wireframe.

>>> TIMEFRAME

1 week to deploy & prepare for presentation

>>> IDEA

Create e-commerce web app using database, stock images of products, and technologies to handle email and tending.

>>> TEAM MEMBERS

Rachael Adams, Will Hearn, C. Samuel Sullivan

>>> ROLE

Mostly sticking to Front-End development, but additionally Navigation/Footer and collaboration throughout were my team contributions.

>>> FEATURES

  • Landing Page
  • Inventory Page (x2)
  • Ability to add to & remove items from cart, with running total
  • Contact page
  • Checkout page securely handling sensitive information + email services on every order
  • Receipt and invoice

>>> APPROACH

Working together as team, divying up to strengths from the start, passing the hat as support was needed. By this point, we were familiar with who was stronger where, but we took opportunity to learn from each other where we each were weak.

>>> TECHNOLOGIES

  • AngularJS, ExpressJS, jQuery, NodeJS, Bootstrap, Stripe, SendGrid, MySQL, Amazon Web Services
